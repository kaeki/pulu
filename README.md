# sssf-project

Chat application with video chat made using Node.js, WebRTC

### Install

> For database connection create file named config.js to config folder
```javascript
module.exports = {
    user: 'userName',
    pwd: 'password',
    addr: 'addressToDatabase',
    port: 'port',
    db: 'databaseName',
};
``` 
> Then run 
    `npm install`


### Run

> To start serve

    `npm start`
